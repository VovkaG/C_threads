Напишете програма, която търси дадена дума във файлове. Когато има повече от един файл, зададен от командния ред, търси във
всички файлове. Търсенето да бъде разпределено посредством създаване на дъщерни процеси или
нишки, като се извършва комункация между нишките (една да предава информация на друга.)


Write a program that searches for a given word in files. When there is more than one file specified by the command line, searches in
all files. To distribute the search by creating child processes or
threads, communicating between threads (one passing information to another.)
